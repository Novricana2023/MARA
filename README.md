# Afadhi – Smart Safari Park Management Dashboard

Afadhi is a React + Vite prototype that simulates an AI-controlled safari park operations centre. It uses mocked data, animated vehicles, attraction zones, and a fully scripted demo scenario suitable for a tourism innovation pitch.

## Getting started

```bash
cd Afadhi
npm install
npm run dev
```

Then open the URL shown in the terminal (by default `http://localhost:5173`).

## Tech stack

- React + Vite
- React Router
- React Leaflet (map and markers)
- TailwindCSS (modern safari-inspired theme)

Everything runs entirely in the browser – there is no backend, no real GPS, and no real payments. All behaviour is simulated using timers and mock data.
